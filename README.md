# 📊 Sales Data Analysis Dashboard

This project analyzes retail sales data using **Python, SQL, and Power BI** to generate business insights and build an interactive dashboard.

The goal of this project is to demonstrate the complete workflow of a Data Analyst:
data cleaning → analysis → visualization → insights.

---

# 🛠 Tools Used

* Python (Pandas, NumPy)
* SQL (MySQL)
* Power BI
* Excel / CSV

---

# 📂 Dataset

The dataset contains sales transactions including:

* Order Date
* Product Name
* Category
* Sales
* Profit
* Discount
* Region
* Segment
* Quantity

---

# 🔄 Project Workflow

## 1. Data Cleaning (Python)

* Loaded raw dataset
* Handled missing values
* Converted date columns into datetime format
* Created new columns:

  * Order Year
  * Order Month
* Checked data types
* Generated summary statistics

---

## 2. Data Analysis (SQL)

Performed business queries such as:

* Total Sales
* Total Profit
* Total Orders
* Sales by Category
* Profit by Region
* Monthly Sales trend
* Top 10 Products by Sales
* Average Order Value

---

## 3. Dashboard (Power BI)

Created interactive dashboard containing:

### KPI Metrics

* Total Sales
* Total Profit
* Total Quantity Sold
* Profit Margin %

### Visualizations

* Monthly Sales Trend
* Sales by Category
* Profit by Category
* Profit by Region
* Sales by Segment
* Top 10 Products by Sales
* Discount vs Profit relationship

### Filters (Slicers)

* Order Year
* Category
* Region
* Segment

---

# 📈 Key Insights

* Technology category generates highest sales.
* West region produces highest profit.
* High discounts often reduce profit.
* Consumer segment contributes highest revenue.
* Sales show growth trend over time.
